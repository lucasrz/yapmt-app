# YAMPT APP

Project manager API for superplayer assessment

## Installation

Use the node package manager [npm](https://nodejs.org/en/download/) to install dependencies.

Run
```
npm i
```

## Usage

Run

```
npm run start
```

Default adress
```
http://localhost:4000/api
```

# Testing

Run
```
npm run test
```

Run
```
npm run test:coverage
```
